# Weather-app
A weather app that fetches real-time weather data using the OpenWeatherMap API. Built with HTML, CSS, and Vanilla JavaScript.

## Features
- Search weather by city name
- Displays temperature, humidity, wind speed, and feels like
- Weather icon from OpenWeatherMap
- Responsive design for mobile and tablets
- Enter key support for faster searching
- Debounced search to avoid unnecessary API calls

## Technologies Used
- HTML
- CSS
- Vanilla JavaScript
- OpenWeatherMap API

## How to Use
1. Clone the repository
2. Get a free API key from [openweathermap.org](https://openweathermap.org)
3. Paste your API key in `weather.js`
4. Open `index.html` in your browser
